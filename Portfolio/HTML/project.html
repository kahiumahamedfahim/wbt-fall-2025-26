Pharmacy Management System

About This Project :

The Pharmacy Management System is a desktop-based application developed using C# (Windows Forms) and SQL Server. It is designed to efficiently manage all pharmacy operations, including user registration, medicine inventory, order processing, and sales tracking with a clean, role-based interface for different types of users.

The system includes two main roles: Admin and Manager

  • Admin can add new admins and managers, update their details, and oversee the entire system.
  • Manager can sell medicine , manage stock and record sales details for tracking and reporting

Tech Stack :

  • Frontend: C# Windows Forms
  • Backend: SQL Server
  • Architecture: Role-based access with modular UserControls and CRUD functionalities

Key Features:

  • Secure login and registration system
  • Admin and Manager role management
  • Medicine sales and record tracking
  • CRUD operations for all data modules
  • Sales report management
  • User-friendly and responsive interface

Book Crud Operation

About the Project:

The Book CRUD Operation is a web-based application developed using ASP.NET Core for the backend and Bootstrap for responsive frontend design. It demonstrates full Create, Read, Update, and Delete (CRUD) functionality for managing a collection of books. The project uses an in-memory list to store book data, making it simple yet effective for learning and demonstration purposes.

Key Features:

  • Add new books with details such as Title, Author, and Price
  • View the list of all books in a responsive table
  • Update existing book details
  • Delete books from the list
  • Responsive and user-friendly interface using Bootstrap

Tech Stack:

  • Backend: ASP.NET Core
  • Frontend: Bootstrap
  • Data Storage: In-memory List
